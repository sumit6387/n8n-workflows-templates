## How to Use a Template

1. **Download or clone this repository**
   ```bash
   git clone https://github.com/sumit6387/n8n-workflows-templates.git 
   ````
- **Open your n8n Editor**
  - Self-hosted n8n
  - n8n.cloud

- **Import the Workflow**
  - Click **Import from File**
  - Select the `.json` workflow file

- **Configure Credentials**
  - Add required API keys, tokens, or accounts
  - Replace any placeholder values

- **Save & Activate**
  - Test the workflow
  - Activate when ready
